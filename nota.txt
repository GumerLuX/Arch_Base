Instalacion Nueva desde cero.

El archivo Arch_Base

Consta de 2 scripts principales.
El primjero instala el sistema base.
Hace una instalacion base con sistema de archivos btrfs y sus carpetas.
El segundo instala la configuracion de usuario y deja el sistema preparado para una instalacion de escritorio.


El script esta sacado de la paguina:
https://gitlab.com/stephan-raabe/archinstall
